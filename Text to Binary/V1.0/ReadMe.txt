Version 1.0

This program allows you to:
*) convert text to binary code.
*) convert binary code to text.
*) convert text to bytes.
*) convert binary code to bytes.

Note: this version is very 'space' sensitive with conversion from binary code.
      after every 6-8 numbers there should be a space.

This software was programmed on Visual Studio Code (.Net 6.0).
Note: This software was a coding exercise, so expect issues with it.
